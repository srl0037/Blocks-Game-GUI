Practice with Qt Graphics View Framework in PySide6 in free time.

Creation of simple game. Allows users to interact with blocks, add block, move blocks, and delete blocks. Addition possible with space bar; delete blocks by double clicking. Can zoom in/out with ctrl+scroll (cmd+scroll for macOS users). Ability to drag mouse and delete multiple specific squares. Can add multiple squares with drop down menu selection options. Can clear all squares or a random selectio of half of the squares. 

Run application by downloading files, entering folder, and running "python3 main.py"
